What is CSS?

Cascading Style Sheets(CSS) is usee to format the layout of a webpage.

With CSS, you can control the color, font, the size of text, the spacing between elements, how elements are positioned an laid out, what background images of background colors are to be used, different displays for different devices and screen sizes, anh much more!

Tip: The word cascading means that a style applied to a parent element will also apply to all children elements within the parent. So, if you se the color of the body text to "blue", all headings,paragraphs, and other text elements within the body will also get the same color (unless you specify something else)!
